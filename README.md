# React MarkdownEditor Example

## Local development setup

```sh
$ node -v
v6.10.2

$ npm -v
3.10.10

$ yarn --version
0.20.3
```

```sh
$ git clone https://github.com/yossy0228/react-markdown-editor-example.git
$ cd react-markdown-editor-example
$ yarn
```

### Install Yarn

```sh
$ npm i -g yarn
$ yarn --version
```

## Usage

```sh
$ yarn start
```
or
```sh
$ npm start
```
